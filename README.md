
## installation guidelines

### Installation dependency
Enter the root directory
```
npm install
```
then
```
npm preinstall
```
### Run
Start the server end in development mode
```
SET NODE_ENV=development& npm run dev:server
```
then start the front end
```
npm run dev:client
```
